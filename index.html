QInspect - Manajemen Dokumen Kualitas

Aplikasi manajemen dokumen kualitas untuk proyek konstruksi dengan OCR terintegrasi Hugging Face

QInspect adalah aplikasi berbasis web untuk mengelola dokumen kualitas proyek konstruksi. Menggunakan teknologi OCR dari Hugging Face, aplikasi ini dapat mengekstrak data dari dokumen scan dengan akurasi tinggi.
